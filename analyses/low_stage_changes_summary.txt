LOW STAGE İÇİN YAPILABİLECEK DEĞİŞİKLİKLER ÖZETİ
================================================================================

MEVCUT DURUM:
- Başlangıç yüksekliği: 5-15m (Low range'inde) ✓
- OutOfBounds: 45m
- Ceiling: 100m
- Tilt terminate: up_y < 0.5
- Spin terminate: abs(wy) > 10.0
- Landing zone (Success): 5.0m
- Success sadece: abs(vy) < 4.0 m/s (yatay hız, spin, tilt kontrolü YOK)

================================================================================
KATEGORİ 1: FİZİKSEL SINIRLAR (Termination) - LOW İÇİN
================================================================================

A) OutOfBounds: 45m → 25m
   - Neden: Low'da agent drift edip gezer, daha erken durdur
   - Etki: Agent merkeze yakın kalmayı öğrenir
   - Risk: Çok sıkı olabilir, test gerekli

B) Ceiling: 100m → 60-75m
   - Neden: Low'da 100m çok yüksek, agent yukarı kaçabilir
   - Öneri: 75m (güvenli orta)
   - Etki: Agent yere odaklanır

C) Tilt terminate: up_y < 0.5 → up_y < 0.35
   - Neden: Agent düzeltmeyi öğrensin (0.5 çok erken)
   - Etki: Daha fazla düzeltme fırsatı
   - Risk: Çok gevşek olursa yan yatıp devam edebilir

D) Spin terminate: abs(wy) > 10.0 → w_mag > 10-12 rad/s
   - Neden: Sadece wy değil, tüm açısal hız vektörü kontrol edilmeli
   - w_mag = sqrt(wx^2 + wy^2 + wz^2)
   - Etki: Daha doğru spin kontrolü
   - Not: 10.0 zaten yakın, sadece w_mag kullanımı değişiklik

E) Soft edge cezası (OutOfBounds için 18m'den sonra)
   - Yeni özellik: Episode bitirmeden uyar
   - Etki: Agent sınırları öğrenir

================================================================================
KATEGORİ 2: SUCCESS KRİTERLERİ - LOW İÇİN GEVŞETME
================================================================================

ŞU AN EKSİK OLANLAR:
- Yatay hız kontrolü YOK
- Spin kontrolü YOK
- Tilt kontrolü YOK
- Sadece dikey hız var

ÖNERİLEN EKLEMELER:

A) Landing zone: 5.0m → 4.0m
   - Neden: Biraz gevşet (low için)
   - Şu an: 5.0m (zaten yakın)
   - Etki: Küçük ama anlamlı gevşeme

B) Yatay hız kontrolü EKLE: v_h <= 2.5 m/s
   - Şu an: YOK
   - Öneri: v_h = sqrt(vx^2 + vz^2) <= 2.5 m/s
   - Etki: Yan yatıp kayarak inişi engeller

C) Spin kontrolü EKLE: w_mag <= 3.0 rad/s
   - Şu an: YOK
   - Öneri: w_mag = sqrt(wx^2 + wy^2 + wz^2) <= 3.0 rad/s
   - Etki: Dönerek inişi engeller

D) Tilt kontrolü EKLE: up_y >= 0.88
   - Şu an: YOK
   - Öneri: up_y >= 0.88 (yaklaşık 28-30° tilt)
   - Etki: Çok yan yatıp inişi engeller
   - Not: 0.88 çok gevşek olabilir, 0.90-0.92 düşünülebilir

E) Dikey hız: abs(vy) < 4.0 m/s → abs(vy) <= 4.0 m/s
   - Şu an: < 4.0 (zaten yakın)
   - Etki: Minimal, sadece edge case

================================================================================
KATEGORİ 3: CRASH/MISSEDZONE AYRIMI
================================================================================

ŞU AN:
- MissedZone: dy<=1 ve dist_h > 5.0m → -300
- Crash: dy<=1 ve zone içi ama vy >= 4.0 m/s → -300

ÖNERİLEN İYİLEŞTİRMELER:

A) MissedZone: dist_h > 4.0m (5.0m yerine, landing zone ile uyumlu)

B) HardCrash ekle: vy < -8.0 m/s → ekstra ceza (-400 veya -500)
   - Neden: Çok sert çakılmaları ayırt et
   - Etki: Agent yavaş inişi öğrenir

C) Crash nedenlerini daha detaylı:
   - Yatay hız çok yüksek → Crash (YatayHiz)
   - Spin çok yüksek → Crash (Spin)
   - Tilt çok kötü → Crash (Tilt)
   - Dikey hız çok yüksek → Crash (DikeyHiz)

================================================================================
KATEGORİ 4: SHAPING (Reward Shaping) - LOW İÇİN
================================================================================

ŞU AN:
- dist_penalty: 0.05 * dist_h
- horizontal_penalty: 0.03 * v_h
- velocity_penalty: karmaşık (dy<15 iken)
- shaping_stab: 0.04 * up_y (up_y > 0.5 iken)
- step_penalty: -0.02

ÖNERİLEN DEĞİŞİKLİKLER:

A) dist_penalty: 0.05 → 0.08
   - Neden: Drift'i daha güçlü cezala
   - Etki: Merkeze yakın kalmaya zorlar

B) horizontal_penalty: 0.03 → 0.06
   - Neden: Yatay hızı daha güçlü cezala
   - Etki: Yatay hızı kontrol eder

C) velocity_penalty: Basitleştir
   - Şu an: Karmaşık (dy<15, formül karmaşık)
   - Öneri: dy < 10 iken -0.15 * abs(vy)
   - Etki: Daha anlaşılır, yere yakınken yavaşlat

D) shaping_stab: 0.04 → 0.05
   - Neden: Dik duruşu daha güçlü ödüllendir
   - Etki: Stabilite öğrenimi

E) Tilt penalty ekle: up_y < 0.85 iken ceza
   - Yeni özellik: -0.02 * (0.85 - up_y) gibi
   - Etki: Dik durmayı öğret

F) Spin penalty ekle: w_mag > 3.0 rad/s iken ceza
   - Yeni özellik: -0.01 * (w_mag - 3.0) gibi
   - Etki: Spin'i kontrol et
