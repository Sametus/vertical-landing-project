DAMPING DEĞER ANALİZİ (Mevcut: Linear 0.07, Angular 1.2)
================================================================================

MEVCUT DURUM:
================================================================================
- Linear Damping: 0.07 (eskiden 0.1 → düşürülmüş)
- Angular Damping: 1.2 (eskiden 1.5 → düşürülmüş)

TRAINING ÇIKTILARINDAN GÖZLEMLER:
================================================================================
- Agent hız kontrolü yapamıyor (ortalama -4.0 m/s)
- MissedZone oranı yüksek (%67)
- Agent hız ve mesafe kontrolünü birlikte yapamıyor
- Progressive velocity penalty henüz etkisini göstermedi

================================================================================
DAMPING ETKİSİ:
================================================================================

LINEAR DAMPING (0.07):
- Her saniyede hızın ~%7'si kaybolur
- Roket sürekli yavaşlıyor (thrust olmasa bile)
- Bu "hava sürtünmesi" etkisi yaratır
- Uzay ortamı için gerçekçi DEĞİL (hava yok)
- Agent hız kazanmakta zorlanır

ANGULAR DAMPING (1.2):
- Açısal hızın sönümlenmesi
- Çok yüksek! Açısal hareket çok çabuk durur
- Roket dönüş yapmakta zorlanır
- Kontrol etmek zorlaşır
- Agent pitch/yaw kontrolü yapmakta zorlanabilir

================================================================================
ÖNERİ: EVET, DÜŞÜRMELİ!
================================================================================

NEDEN:
1. Uzay ortamı simülasyonu: Hava sürtünmesi olmamalı
2. Agent kontrol zorluğu: Yüksek damping → daha zor kontrol
3. Training performansı: Düşük damping → daha kolay öğrenme
4. Gerçekçilik: Roketler uzayda çalışır, damping minimal

ÖNERİLEN DEĞERLER:
================================================================================

LINEAR DAMPING:
- Şu an: 0.07
- Öneri: 0.01 - 0.02 (çok düşük)
- Alternatif: 0.0 (tamamen kaldır)
- Etki: Roket artık sürekli hız kaybetmeyecek
        Sadece thrust ve gravity etkisi olacak

ANGULAR DAMPING:
- Şu an: 1.2 (ÇOK YÜKSEK!)
- Öneri: 0.1 - 0.3 (çok düşük)
- Alternatif: 0.05 - 0.1 (daha da düşük)
- Etki: Açısal hareket daha serbest
        Agent pitch/yaw kontrolü daha kolay

================================================================================
BEKLENEN ETKİLER:
================================================================================

POSITIVE (İYİ):
- Roket daha hızlı hareket edebilecek
- Agent kontrolü daha kolay
- Daha gerçekçi fizik (uzay ortamı)
- Hız kontrolü daha doğal (sadece thrust/gravity)
- Angular kontrol daha responsive

RİSK (DİKKAT):
- Kontrol çok zorlaşabilir (damping çok düşükse)
- Agent başlangıçta daha fazla başarısız olabilir
- Ama uzun vadede daha iyi öğrenecek
