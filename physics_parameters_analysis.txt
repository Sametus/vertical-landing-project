2. THRUST ANALİZİ (17000 N):
================================================================================

FİZİKSEL HESAPLAMALAR:
- Mass: 1000 kg
- Thrust: 17000 N
- Weight: 1000 * 9.81 = 9810 N
- Net kuvvet (yukarı): 17000 - 9810 = 7190 N
- Net ivme (yukarı): 7190 / 1000 = 7.19 m/s²

MAXİMUM DÜŞÜŞ HIZI (Teorik):
- 15m serbest düşüş: v = sqrt(2*g*h) = sqrt(2*9.81*15) ≈ 17.15 m/s
- Bu hızı sıfırlamak için:
  - Net ivme: 7.19 m/s² (yukarı)
  - Zaman: t = 17.15 / 7.19 ≈ 2.4 saniye
  - Mesafe: s = 0.5 * 7.19 * 2.4² ≈ 20.7 m
  - Yani 15m'den düşerken son 20m'de fren yapmak gerekir (mantıksız!)

THRUST DEĞERLENDİRMESİ:
- Önceki: 15000 N
- Mevcut: 17000 N (artırılmış, iyi!)
- İdeal: 20000 - 25000 N (daha güçlü motor)
- 17000 N yeterli mi? → EVET, ama daha güçlü olursa daha iyi

SONUÇ: Thrust 17k yeterli ama 20k-22k yapılabilir. Öncelik düşük değil.

3. RCS ANALİZİ (1200 N):
================================================================================

RCS (Reaction Control System):
- Açısal kontrol için (pitch, yaw, roll)
- Thrusters roket gövdesine dik açıyla yerleştirilmiş
- Tork üretir: τ = F * r (r = moment arm, yaklaşık 2-3m)

FİZİKSEL HESAPLAMALAR:
- RCS Force: 1200 N
- Moment arm: ~2.5 m (varsayım)
- Tork: 1200 * 2.5 = 3000 N·m
- Moment of inertia (I): ~5000 kg·m² (tahmin, silindirik roket)
- Açısal ivme: α = τ / I = 3000 / 5000 = 0.6 rad/s²

AÇISAL KONTROL HIZI:
- 0.6 rad/s² ile 1 rad/s hızına ulaşmak için: t = 1 / 0.6 ≈ 1.7 saniye
- Bu MAKUL ama yavaş olabilir

RCS DEĞERLENDİRMESİ:
- 1200 N → İyi başlangıç
- Öneri: 1500 - 2000 N (daha responsive kontrol)
- Ama önce damping'i düşür, sonra RCS'yi test et

SONUÇ: RCS 1.2k yeterli ama 1.5k-2k yapılabilir. Öncelik orta.
